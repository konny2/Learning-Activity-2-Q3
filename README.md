===============================================================
PROJECT OVERVIEW
===============================================================
This project examines whether offering a 10% promotional discount has a statistically significant impact on customer purchase behavior. A synthetic A/B testing dataset consisting of 200 customer transactions was generated and divided equally into two independent groups: Group A (No Discount) and Group B (10% Discount). Python was used to simulate the data, compute descriptive statistical summaries, generate analytical visualizations, formulate statistical hypotheses, and perform Welch’s Independent Two-Sample T-Test.

The main objective of the project is to determine whether customers who receive the discount treatment spend significantly more than customers who receive no discount.

Decision:
